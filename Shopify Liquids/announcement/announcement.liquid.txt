{%- comment -%}
  ============================================================================
  AVRO — ANNOUNCEMENT BAR  (Shopify Online Store 2.0 section)
  ----------------------------------------------------------------------------
  HOW TO USE
  1. Create sections/avro-announcement.liquid and paste this entire file in.
  2. Add it to your header group (sections/header-group.json) ABOVE the
     avro-header section, or render it in layout/theme.liquid above the header.
  3. Editable in the theme editor:
       - Each scrolling message is an add / remove / reorder BLOCK (with
         default AVRO messages pre-filled so it always shows something).
       - The "Shop" pill label + link.
       - Scroll speed (seconds per loop).
       - Color mode: "Dark-mode paths" switches the bar to the deep-black +
         gold Zero-Proof palette on matching pages (default: the Social page),
         matching the header. Leave blank to keep it light everywhere.
  This bar scrolls WITH the page (it is not sticky); only the nav sticks.
  No images / assets required.
  ============================================================================
{%- endcomment -%}
